Rules before Starting:

  • User places four ships of length 4,3,2,1 respectively.
  • To place ship user click on desired starting block(in GRID 1).
  • A user can only place vertical ships in seperate columns.
  • If user tries to place ships in similar columns it throws an error.
  • User will play in the second GRID.
  • If user's hit is successful then it will be displayed as green color and a miss hit will be displayed as pink color.
  • Game is available for desktop version
  • Press "Next" to play the Game.
